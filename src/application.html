Session Password

Enter a password in the field below to start a new session. All passwords are valid to Plauzible but only the password that was used to create an entry can be used to view it later.

A password must be provided and must be a minimum of 10 characters in length.

Note: To maximize security you should avoid using common or obvious passwords and make any passwords you choose to use as long as possible. Plauzible requires session passwords to be a minimum of 10 characters in length.

Encryption Settings

Your encryption settings contain details for parameters used to encrypt your records. This is sensitive information and should not be shared with anyone. If you wish to use Plauzible on more then one machine then you can must copy these details between the machines manually.

Warning: Your encryption settings impact how your records are encrypted. If you change these settings then you will lose access to the records you previously created on this machine! If you have not yet created any records then you can safely change these settings.
