Welcome To Plauzible

It looks like this is the first time that you've run the application on this machine. Before you can start using the application there are a number of set up steps that we need to don first. Click on the continue button to start this process.

Encryption Settings

The first step in the process is to generate an encryption salt for this installation. If you have no idea what this is just leave the field below empty and the application will automatically generate a salt for you. If you plan on using Plauzible across multiple machines then this value needs to be the same on all machines that you want to use, paste the value you want this installation to use into the field below.

Note: Your encryption salt is considered sensitive information and you should not share it with anyone.

When you're ready click on the next button below.

Online Service Key

If you want to easily use Plauzible across multiple machines then you can store your records in an online service. Encryption of these records will still occur locally but the encrypted data will be stored on a networked server for retrieval across platforms. Using an online service requires an access key which you should enter in the field below.

If you do not want to store your data with an online service or only wish to use Plauzible on the current platform you can leave the field below empty.

When you're ready click on the next button below.
